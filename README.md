# Swiggy Sales Dashboard

## Project Overview

This excel dashboard provides a comprehensive analysis of Swiggy sales performance across different states, food categories, time periods, and customer ratings. The dashboard helps identify sales trends, customer preferences, and business performance through interactive visualizations.

##Dataset used
<a href=" https://github.com/sumitup43/Swiggy-Sales-Dashboard/blob/main/swiggy%20sales%20analysis.xlsx"> dataset </a>
## Business Insights - 
Identify top-performing states and cities. - Monitor sales trends across different time periods. - Compare food category performance. - Track customer satisfaction through ratings. - Analyze order volume and revenue generation.
